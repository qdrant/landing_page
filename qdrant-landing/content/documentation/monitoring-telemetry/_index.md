# Monitoring & Telemetry

Qdrant exposes metrics in Prometheus/OpenMetrics format, making it straightforward to integrate with standard observability stacks. This section covers metrics endpoints for self-hosted deployments as well as step-by-step guides for setting up Prometheus and Grafana in cloud environments.

## Guides

- [Monitoring & Telemetry](/documentation/monitoring-telemetry/monitoring/) — Explore available metrics endpoints, node-level metrics, and health check endpoints exposed by Qdrant.
- [Managed Cloud Prometheus Monitoring](/documentation/monitoring-telemetry/managed-cloud-prometheus/) — Set up Prometheus and Grafana to monitor Qdrant databases running in Qdrant Managed Cloud.
- [Self-Hosted Prometheus Monitoring](/documentation/monitoring-telemetry/hybrid-cloud-prometheus/) — Set up Prometheus and Grafana to monitor Qdrant running in Hybrid Cloud or Private Cloud Kubernetes deployments.
