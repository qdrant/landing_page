# Monitoring & Telemetry

Monitor, observe, and collect telemetry from your Qdrant deployment — whether self-hosted or running on Qdrant Cloud.

- **[Monitoring & Telemetry](/documentation/monitoring/monitoring/)** — Metrics endpoints, Prometheus/OpenMetrics integration, telemetry, and Kubernetes health checks.
- **[Managed Cloud Prometheus Monitoring](/documentation/monitoring/managed-cloud-prometheus/)** — Set up Prometheus and Grafana to monitor Qdrant databases running in Qdrant Managed Cloud.
- **[Self-Hosted Prometheus Monitoring](/documentation/monitoring/hybrid-cloud-prometheus/)** — Set up Prometheus and Grafana to monitor Qdrant databases in Hybrid or Private Cloud Kubernetes clusters.
