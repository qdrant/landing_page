Update the collection with patch requests to move vector data to the `cold` memory tier. This functionality allows you to specify vector parameters, such as setting a vector's memory tier, for a specific collection. In cases where the collection does not have named vectors, an empty name (`""`) should be used to perform this update. This feature provided by Qdrant version 1.4 enables modifications to collection parameters, including HNSW index, quantization, and memory tier configurations dynamically, without the need to recreate the collection. Additionally, segments containing index and quantized data will be automatically rebuilt in the background to align with the new parameters.
